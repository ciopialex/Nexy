Firebase Configuration Missing!

It looks like you haven't added your Firebase configuration keys to the index.html file yet.

Please follow these steps:

  1. Go to your Firebase project console.
  2. Navigate to Project Settings (click the ⚙️ icon).
  3. In the "Your apps" section, find your web app.
  4. Copy the firebaseConfig object.
  5. Paste it into the index.html file where indicated.

The app will not work until this is done.
